# *** ONE STOP GUIDE TO INSTALL AND CONFIGURE PUPPET ***

## Make sure you follow the sequence listed below:

### First open slave machine and run the following:
```
chmod +x slave.sh
sudo ./slave.sh

```

### Now open the master machine and run the following:

```
chmod +x master.sh
sudo ./master.sh

```
